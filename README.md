# Final Project: Group 50 [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/nqh2pitt/CMPINF-11-Final-Project/HEAD?urlpath=%2Fdoc%2Ftree%2FFinalCombined.ipynb)

This project analyzes parks, traffic, and dumpsite data to identify which neighborhood has the strongest overall infrastructure.

Group members:
- Alex Jin
- Nick Hoffman
- Jared Russman

Datasets Used:
- Dumpsites: [csv file](https://data.wprdc.org/dataset/ed4cf849-d323-4d01-8a13-675d283f1ac9/resource/ee834d8d-ae71-4b3b-b02b-312ba321ff17/download/illegaldumpsites.csv)
  - This dataset has been used to calculate the amount of sites in every neighborhood and how many tons of garbage in each area
- Monthly OTP by Route: [csv file](https://data.wprdc.org/datastore/dump/00eb9600-69b5-4f11-b20a-8c8ddd8cfe7a)
  - This dataset was used to calculate the on-timeness for each route in a neighborhood.
